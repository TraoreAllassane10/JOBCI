# JOBCI

## Presentation 
JOBCI est une plateforme de recherche d'emploi et de mini-emploi dans les domaines tel que :
- Service domestique (Nounou, menagère, servante, etc.)
- Service personnel (Coiffeur, Chauffeur, etc.)
- Artisanat et technique (Plombier, mason, peintre, etc.)
- Evenementiel (DJ, Serveur, etc.)
- Commerce et gestion (Gerant de cyber, caisière, vendeur)

## Fonctionnalitées 
1. Recruteur
- Authentification
- Gestion des offres
- Gestions des candidatures

2. Chercheur d'emploi 
- Authentification 
- Recherche d'offres
- Candidature
- Favorise des offres

## Technologies
1. Frontend
- React 
- Redux toolkit
- Axios
- Tailwindcss

2. Backend
- Laravel
- Laravel Sanctum
